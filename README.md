## âš™ï¸ How It Works (Step-by-Step)

1. The user enters a task in the input field.
2. JavaScript captures the form submission event.
3. A new task item is dynamically created and added to the task list.
4. The task is saved to Local Storage to ensure persistence.
5. On page reload, tasks are retrieved from Local Storage and displayed automatically.
